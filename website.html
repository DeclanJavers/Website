Personal Website

I am happy to have just released my first large CS project, my own website.


Over the summer of 2023, I wanted to learn more about websites and how they are developed, so I set out to build my own. My primary goal was to create a professional-looking website that showcased my ability to develop CS skills. Up to this point, I had not taken any college courses in html, css, or javascript, so I found that this was a great way for me to have the chance to advance my skills and learn on my own.

My first step in creating this website was to research the skills needed to create it. I settled on this Youtube Course to teach me the foundations of html, css, and the javascript implementation in the html code. I found I learn very well when I have the chance to watch a course and implement the techniques taught at the same time. Ultimately, I am proud to have achieved a solid foundation on html, css, and javascript and their use in a website.

I also learned about how to set up a larger scale project. The importance of pre-planning everything before coding it was clearly demonstrated to me. For the website design, I used Adobe Illustrator to create an appealing design that would also be functional and easy to implement. The design I came up with was simple, yet I think it looks really good. The design feature that I am most proud of is the background, which is a stylized map of Boston, which I think looks cool and conveys a unique feeling.

Overall, building my website has been a great experience. I have gained the foundations in html, css, and javascript and I had a great time learning to implement them. I was able to express myself through my design and, in the end, I created a personal website that I am proud of.
